# Laserstrike

A first-person shooter game where survival is key. Face off against auto-aiming turrets shooting lasers from every direction. Destroy them all or fall to their relentless assault.

## Overview

Laserstrike is a fast-paced first-person shooter developed as a Computer Graphics project. Players must navigate hostile environments while eliminating intelligent turret enemies that track and fire upon them. The game features immersive graphics techniques including real-time lighting, dynamic shadows, and particle effects.

## Key Features

- **Immersive First-Person Perspective** - Smooth camera controls with mouse look system
- **Intelligent Enemy AI** - Auto-aiming turrets with pathfinding capabilities
- **Dynamic Health System** - Visual vignette effects that intensify as you take damage
- **Realistic Physics** - Physics-based interactions and ragdoll deaths
- **Advanced Graphics** - PBR materials, custom shaders, and post-processing effects
- **Particle Systems** - Bullet trails, explosions, and impact effects

## Objective

Eliminate all turret enemies in each level to achieve victory. Avoid their laser fire and survive the onslaught to complete the game and achieve high scores.

## Technical Highlights

### Graphics Techniques

- **Lighting System**: Directional lighting with real-time shadow casting and cascaded shadows
- **Camera Transformations**: First-person view with yaw/pitch controls, head bob, and FOV adjustments
- **Texture Mapping**: Diffuse, normal, and PBR materials with metallic/specular maps
- **Custom Shaders**: Weapon shaders with Fresnel effects, emission mapping, and post-processing
- **Animation Systems**: State machine-based animations with IK for foot placement and layered animation support

### Technologies Used

- **Unity Game Engine** - 3D game development and rendering
- **C#** (98.6%) - Game logic and mechanics
- **Git/GitHub** - Version control
- **Photoshop** - Texture creation
- **Unity Asset Store** - Additional assets

## Getting Started

### For Developers

**Prerequisites:**
- Unity Hub
- Unity 2021.3 LTS (or compatible version)
- Git

**Installation:**

1. Clone the repository:
```bash
git clone https://github.com/ammarIbrahim59/fps-game.git
```

2. Open Unity Hub and add the project

3. Open the main scene from the `Scenes` folder

4. Click the Play button to run the game

### For Players

1. Download the latest release from [GitHub Releases](https://github.com/ammarIbrahim59/fps-game/releases)

2. Extract the ZIP file to your desired location

3. Run the executable:
   - **Windows**: `FPSGame.exe`
   - **macOS**: `FPSGame.app`

4. Configure graphics settings on first launch as needed
